# Walkthrough: Feature Pages Round 5 Fine-Tuning

## Key Changes

### 1. 地圖連動優化 (住宿資訊)
- **精準定位**：現在點擊住宿資訊頁面的 **MAP 按鈕**，不再自行建立新標籤，而是直接連動至「行程清單」Day 1 的 **「飯店 Check-in」** 事件。
- **自動切換**：若當前不在 Day 1，系統會自動切換回 Day 1 並聚焦到正確的飯店位置。

### 2. 標頭地圖按鈕狀態修復
- **狀態同步**：修復了右上角地圖 Icon 的視覺狀態。開啟地圖時會顯示 **反灰底色 (Active State)**，關閉後則會正確 **恢復為初始樣式**。

### 3. 幼兒頁面介面細節
- **高度統一**：「哭鬧處理建議」的方框高度已進行修正，現在與「作息注意事項」保持一致，視覺感更清爽。
- **排版優化**：優化了「幼兒用品採買」店家的標題與內文間距，並確保所有資訊都是頂端對齊。
